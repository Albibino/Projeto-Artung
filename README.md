# Projeto Artung

## Conceito

- Ser uma plataforma para divulgação de artes, para artistas underground até os artistas mainstreams.
- Sistema de divulgação de novos artistas.

## Especificações

- Desenvolvido em PHP utilizando a framework Laravel
